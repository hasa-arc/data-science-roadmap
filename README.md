# 📊 ডেটা অ্যানালাইসিস শেখার রোডম্যাপ

> **বিজ্ঞান প্রযুক্তির তৈরি — শূন্য থেকে এক্সপার্ট পর্যন্ত**

এই রিপোজিটরিটি ডেটা সায়েন্স এবং ডেটা অ্যানালাইসিস শেখার জন্য একটি সম্পূর্ণ, ধাপে ধাপে রোডম্যাপ। এখানে Kaggle, IBM Cognitive Class, MATLAB Academy এবং HP LIFE-এর ফ্রি কোর্সগুলো সংগ্রহ করে সাজানো হয়েছে।

## 🚀 GitHub Pages-এ পাবলিশ করার নিয়ম

১. GitHub-এ একটি নতুন রিপোজিটরি তৈরি করুন (যেমন: `data-science-roadmap`)
২. এই `index.html` ফাইলটি রিপোজিটরির root-এ আপলোড করুন
৩. **Settings → Pages → Branch: main → Save** ক্লিক করুন
৪. কিছুক্ষণ পর আপনার সাইট লাইভ হবে: `https://yourusername.github.io/data-science-roadmap`
